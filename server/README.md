## 友得云客服务端安装指南

请参照如下步骤进行操作，部署友得云客服务端：

1. **服务端配置文件准备**
使用提供的`docker-compose.yml`文件作为友得云客服务端的基础配置文件。该文件包含了容器化部署所需的所有服务及其依赖关系的详细配置信息。

2. **参考官方文档**
请访问官方文档网址以获取详细的安装指导：[https://doc.youdeyunke.com/zhunbei.html](https://doc.youdeyunke.com/zhunbei.html)。

确保在安装过程中遵循官方指南，确保服务端的稳定运行与数据安全。如遇问题，请查阅文档中的故障排查部分或联系友得云客技术支持获取帮助。
